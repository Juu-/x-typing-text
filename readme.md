## Usage
```html
<x-typing-text text="plop"
                 waiting-time="500"
                 delay="1000">
  </x-typing-text>
```
### Inputs:
```text : the text who will be typing```
```waiting-time: the delay between each character```
```delay: the delay to start typing```
